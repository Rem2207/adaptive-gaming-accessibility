Adaptive Gaming Accessibility

Middleware de accesibilidad para videojuegos orientado a mejorar la interacción de usuarios con diferentes capacidades mediante tecnologías adaptativas.

🇪🇸 Español
Descripción

Adaptive Gaming Accessibility es una aplicación desarrollada para mejorar la accesibilidad en videojuegos mediante la adaptación de los métodos de entrada tradicionales.

El sistema permite:

Reconocimiento gestual mediante cámara.
Mapeo de controles adaptativos.
Soporte para controladores Xbox.
Emulación de teclado y mouse.
Alertas visuales para eventos sonoros.
Configuración personalizada de dispositivos.
Interfaz multilenguaje.
Tecnologías utilizadas
Python
Tkinter
OpenCV
MediaPipe
Pygame
Pynput
Instalación
pip install -r requirements.txt
python main.py
Estado del proyecto

Proyecto académico desarrollado para la carrera de Ingeniería en Informática de la Universidad de Palermo.

Description

Adaptive Gaming Accessibility is an accessibility middleware designed to improve video game interaction through adaptive technologies.

The system provides:

Camera-based gesture recognition.
Adaptive controller mapping.
Xbox controller support.
Keyboard and mouse emulation.
Visual audio alerts.
Custom device configuration.
Multilanguage interface.
